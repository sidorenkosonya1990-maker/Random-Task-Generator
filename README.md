# Random Task Generator (Генератор случайных задач)

## Автор
Сидоренко Софья

## Описание
GUI-приложение для генерации случайных задач из предопределенного списка. Приложение позволяет:
- Генерировать случайные задачи по категориям (учёба, спорт, работа)
- Добавлять собственные задачи
- Отслеживать историю сгенерированных задач
- Фильтровать историю по типу задач
- Сохранять историю в JSON формате

## Требования
- Python 3.6 или выше
- Библиотека tkinter (обычно входит в стандартную установку Python)
- Библиотека random (стандартная библиотека Python)
- Библиотека json (стандартная библиотека Python)
